# Portfolio — Baijou Mouad

## Structure des Fichiers

```
portfolio/
├── index.html     → Page principale du portfolio
├── style.css      → Feuille de styles (design complet)
├── script.js      → Interactions JavaScript
├── photo.jpg      → (À ajouter) Votre photo de profil
└── README.md      → Ce fichier
```

## Comment utiliser

### 1. Ajouter votre photo
- Renommez votre photo en `photo.jpg`
- Placez-la dans le même dossier que `index.html`
- Format recommandé : portrait 4/5, minimum 400x500px

### 2. Ouvrir localement
- Double-cliquez sur `index.html` pour ouvrir dans votre navigateur
- Ou utilisez un serveur local : `python -m http.server 8000`

### 3. Personnalisation rapide
Modifiez dans `index.html` :
- Numéro de téléphone
- Email
- Adresse LinkedIn
- Toute information de contact

### 4. Mise en ligne (Hébergement gratuit)
Options recommandées :
- **Netlify** : Glissez-déposez le dossier sur netlify.com
- **GitHub Pages** : Uploadez sur GitHub et activez Pages
- **Vercel** : Connectez votre repo GitHub

## Design

- **Thème** : Navy profond + Or élégant
- **Police** : Playfair Display (titres) + DM Sans (texte)
- **Animations** : Reveal au scroll, barres de langue, effets hover
- **Responsive** : Mobile, tablette et desktop
